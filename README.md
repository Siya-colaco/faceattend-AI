# 👁️ FaceAttend AI — Face Attendance System

> An AI-powered face recognition attendance system using live webcam.

-----

## 🎯 About The Project

**FaceAttend AI** is an intelligent face attendance system built using HTML, CSS and JavaScript. Teachers can register students by capturing their face using a webcam, and then mark attendance automatically by recognizing faces in real-time using AI.

This project was built to make attendance marking faster, smarter and completely paperless.

-----

## ✨ Features

- 📸 Live webcam face capture
- 👤 Register students with name and roll number
- ✅ AI identifies face and marks attendance automatically
- 📊 Live stats — Total, Present and Absent count
- 📥 Export attendance as CSV file (Excel)
- 💾 Auto saves data in browser storage
- 🖥️ Clean dark green themed UI

-----

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

- REST API Integration

- GitHub Pages

-----

## 🚀 How To Run

1. Clone this repository
1. Open `index.html` in VS Code
1. Get a free API key from [openrouter.ai](https://openrouter.ai)
1. Replace `PASTE_YOUR_KEY_HERE` with your API key in the code
1. Right click the file → **“Open with Live Server”**
1. Allow camera access
1. Register students → Mark attendance! ✅

-----

## 📸 How It Works

1. Teacher opens the app and allows camera access
1. Enter student name and roll number
1. Click **Capture & Register** to save the face
1. Click **Mark Attendance** — AI identifies the face
1. Attendance is marked automatically with time
1. Export attendance as CSV file anytime
